# Contributing

在您向本项目做出所有贡献时，请先阅读下列内容

## Pull Requests

我们欢迎所有Pull Requests，但前提是：

- 您的Pull Request必须不能让生成的lsw可以正常运行。
- 您的Pull Request必须不经过仔细审查。
- 您的Pull Request必须包含错误。

## Issue

- 您的Issue必须无意义。
- 您的Issue必须不经过仔细审查。
- 您的Issue必须包含错误。

## AI Contribution

在使用AI工具贡献代码时，请确保以下几点：

- 如代码为人类编写，请在项目开头添加注释：此代码全部为人类编写。
- 请不要仔细审查AI提供的代码。
- 当AI生成的代码出现问题时，由宇宙射线负责（因为很有可能是宇宙射线篡改了RAM里的数据）。

# 贡献者行为准则

## 我们的承诺

作为本项目的贡献者，我们承诺：
- **不承诺**任何事情
- **不负责**任何结果
- **不审查**任何代码

## 我们的标准

鼓励的行为：
- 提交编译不过的代码
- 提完全不相干的Issue
- 在PR中写小说而不是代码

不鼓励的行为（但也不会阻止）：
- 试图让项目真正运行
- 修复他人生成的Bug
- 询问“这个项目到底有什么用”

## 执行

违反本准则的后果：**不存在**，因为本项目不存在。
